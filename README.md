# 九阳炼目 · 视力追踪

每日视力检测记录工具。

## 使用方法

直接打开 [网页链接](https://zhao1427928672-png.github.io/-/) 即可使用。

## 功能

- 👤 多用户管理：2-3 人各自记录，数据带用户标签
- 📝 每日记录：日期、时间、距离、设备、戴镜、天气、光线、练眼情况
- 📊 视力选择：五分记录法 4.0-5.3，支持双眼/左右单测
- 😌 状态记录：眼疲劳、清晰感、测后感觉、练眼体感
- 📋 历史记录：按用户筛选，查看所有记录
- 📈 趋势图表：折线图 + 统计摘要，多用户对比
- 💾 数据导出/导入：JSON 格式，方便备份和共享

## 技术栈

纯前端 HTML + JavaScript，数据存储在浏览器 localStorage，无需服务器。
